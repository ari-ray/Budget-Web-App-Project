# Budget-Web-App-Project

A web-based budget and finance tracking application built using Flask, HTML, CSS and JavaScript.

This is just the prototype, more features to be added soon.

## Features
- Track income and expenses by category
- View totals and net balance
- Interactive charts using Chart.js
- Responsive UI with transitions

## Tech Stack
- Python (Flask)
- HTML, CSS
- JavaScript
- Chart.js

## Purpose
Built as a personal project to demonstrate full-stack fundamentals and UI interactivity.

## Planned Improvements
- Database integration for storage
- Input validation and error handling
- Tests files for core logic
- Downloadable CSV of financial data
- Improved accessibility and UI/UX enhancements
- Deployment
